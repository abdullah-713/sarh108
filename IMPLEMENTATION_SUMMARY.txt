🚀 سادساً: خطوات النشر

الخطوة 1: مسح الـ Cache (اختياري)
  php artisan cache:clear
  php artisan view:clear
  php artisan config:clear

الخطوة 2: بناء جديد
  npm run build
  Result: ✓ built in ~14s

الخطوة 3: مسح متصفح الـ Cache
  Ctrl + Shift + Delete (All time)
  أو Cmd + Shift + Delete للـ Safari

الخطوة 4: تحديث الصفحة
  Ctrl + F5 (Windows/Linux)
  Cmd + Shift + R (Mac)

الخطوة 5: التحقق
  F12 → Network Tab → check app-*.css
